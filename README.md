# Анализ бюджета класса 6В

Веб-приложение для анализа бюджета класса с визуализацией данных о поступлениях, расходах и прогнозах.

## Описание

Проект представляет собой одностраничное веб-приложение, созданное для родителей класса 6В. Содержит:
- Статистику по поступлениям в бюджет класса
- Данные по расходам (ретроспектива и текущие значения)
- Прогноз расходов на оставшиеся учебные четверти
- Информацию об остатках средств на счету каждого ученика
- Предлагаемые дифференцированные взносы

## Технологии

- HTML5
- CSS3
- Адаптивный дизайн

## Использование

Откройте `index.html` в браузере для просмотра.
